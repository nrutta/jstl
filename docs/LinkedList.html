Class: LinkedList

LinkedList

new LinkedList

LinkedList. This is actually implemented as a doubly-linked-list to be efficient when performing operations that require traversing from the end of the list. While there are some methods to support adding and removing elements explicitly at the front and back of the list, this doesn't go so far as to provide all of the functionality to implement a stack or queue (which are provided by separate datastructures). This may turn out to be a short-sided decision.
